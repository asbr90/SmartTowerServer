ZigBee

The wireless control market has a number of unique needs for which ZigBee is ideally suited, because ZigBee is
  • Highly reliable
  • Cost-effective
  • Able to achieve very low power
  • Highly secure
  • An open global standard
  • Low data rate
Based on the IEEE 802.15.4 ZigBee standards of the standard describes the ISO / OSI layers above the data link layer
ZigBee achieves high reliability in a number of ways:
  • CSMA-CA
  • 16-bit CRCs
  • Acknowledgments at each hop
  • Mesh networking to find reliable route
  • End-to-end acknowledgments to verify data made it to the destination
ZigBee enhances reliability through mesh networking, acknowledgments and use of the robust IEEE 802.15.4 standard
ZigBee devices can achieve 5 years on a pair of AA batteries.
ZigBee Alliance membership is required in order to ship ZigBee technology in products. Alliance membership provides early access to specifications.
ZigBee covers many markets, including home, commercial and industrial automation, medical and location-based services.
A ZigBee Coordinator (ZC), which is used to form the network, a ZigBee Router (ZR) which can route packets, and a ZigBee End-Device (ZED), which can sleep and go into low-power modes.
ZigBee Architecture

Addressing within ZigBee includes all of the following components:
  • PAN ID (MAC)
  • NwkAddr (NWK)
  • Endpoint (APS)
  • Profile ID (APS)
  • Cluster (APS)
  • Command and/or attribute (ZCL)
